## 常用熱鍵

1. 全部儲存： 
- [Win] `ctrl + k, s`
- [Max] `cmd + option + s`
2. 切換關聯檔案
- [Win] `alt + o`
- [Max] `option + shift + o`
3. 快速開啟檔案
- [Win] `ctrl + p`
- [Max] `cmd + p`
4. 開啟/關閉左邊的 siderbar
- [Win] `ctrl + b`
- [Max] `cmd + b`
5. 開啟/關閉檔案總管
- [Win] `ctrl + shift + e`
- [Max] `cmd + shift + e`
6. 開啟/關閉搜尋視窗
- [Win] `ctrl + shift + f`
- [Max] `cmd + shift + f`
7. 開啟/關閉原始檔控制(git)
- `ctrl + shift + g`

## 在 Google Chrome 中除錯

1. 按下 `F12`
2. 切換到 Source 頁籤
3. 使用 ctrl + p 搜尋要除錯的 `*.ts` 檔案
4. 插入中斷點
5. 右邊的 watch 可以加入要監看的變數
